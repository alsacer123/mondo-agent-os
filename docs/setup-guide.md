# 安装与使用指南

## 第一步：复制模板

把 `templates/` 复制到你的 Obsidian vault 或普通项目目录。

建议先保留这些文件：

- `_索引.md`
- `_项目规则.md`
- `_总览.md`
- `_当前状态.md`
- `_next.md`
- `决策日志.md`
- `行动池.md`
- `每日笔记.md`
- `AGENTS.md`

## 第二步：建立第一个项目

```text
30_Projects/
  我的第一个AI赋能项目/
    _项目规则.md
    _总览.md
    _当前状态.md
    _next.md
    决策日志.md
```

先不要整理全部资料，只写清：

- 这个项目要解决什么真实问题。
- AI 进入项目时不能做什么。
- 当前唯一下一步是什么。

## 第三步：给 AI 的启动提示

可以这样说：

```text
请先读取工作区根目录的 _索引.md，再读取当前项目的 _项目规则.md 和 _next.md。
只处理这个项目，不读取无关目录。
完成后把结果写回 _next.md；如果有重大取舍，写入 决策日志.md。
```

## 第四步：每天收口

每天结束前做 3 件事：

1. 把完成动作归档到项目 `_next.md`。
2. 把新状态写入 `_当前状态.md`。
3. 把跨项目事项回流到 `行动池.md`。

## 第五步：沉淀方法

当同一种问题出现 3 次，就把它变成模板或 SOP。

例如：

- 项目启动清单
- AI 赋能诊断表
- 客户访谈问题
- 交付验收清单
- 内容选题评分表
